API SummitQuizz

Bienvenue sur l'API de SummitQuizz ! Voici la liste des actions possibles via cette API. Chaque appel renvoi un JSON avec un status (status), une erreur (error) et un résultat (response).

  • Récupérer tous les quizz : /api-quizz/quizz
  • Récupérer un quizz via son id: /api-quizz/quizz/:id
  • Récupérer tous les utilisateurs : /api-quizz/user
  • Récupérer un utilisateur via son id: /api-quizz/user/:id
  • Récupérer tous les RH : /api-quizz/rh
  • Récupérer un RH via son id: /api-quizz/rh/:id
