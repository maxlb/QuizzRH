Bienvenue sur l'API de SummitQuizz !

Voici la liste des actions possibles via cette API. Chaque appel renvoie un JSON avec un status (status), une erreur (error) et un résultat (response).

Quizz

  • api-quizz/quizz : Récupérer tous les quizz
  • api-quizz/quizz/:id : Récupérer un quizz via son id

Joueurs

  • api-quizz/user : Récupérer tous les utilisateurs
  • api-quizz/user/lastActive : Récupérer tous les utilisateurs ayant répondu à un quizz le mois dernier
  • api-quizz/user/:id : Récupérer un utilisateur via son id

RH

  • api-quizz/rh : Récupérer tous les RH
  • api-quizz/rh/:id : Récupérer un RH via son id
